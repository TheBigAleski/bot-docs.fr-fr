# <a name="python-samples-for-bot-framework-sdk"></a>Exemples Python pour le kit SDK Bot Framework
[!INCLUDE [pre-release-label](../includes/pre-release-label.md)]

Ces exemples montrent comment les bots centrés sur les tâches vous aident à tirer parti des fonctionnalités du kit SDK Bot Framework pour Python. Le kit SDK v4 est disponible en préversion. Pour plus d’informations, consultez le [dépôt GitHub](https://github.com/Microsoft/botbuilder-python) Python. 

Pour obtenir les exemples, clonez le dépôt GitHub [botbuilder-python](https://github.com/Microsoft/botbuilder-python) à l’aide de Git.

```cmd
git clone https://github.com/Microsoft/botbuilder-python.git
cd samples
```
Les exemples de bots générés avec le kit SDK Bot Framework pour Python sont organisés dans le répertoire **samples**.
