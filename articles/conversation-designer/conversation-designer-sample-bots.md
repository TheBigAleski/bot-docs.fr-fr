# <a name="create-a-conversation-designer-bot-from-a-sample-bots"></a>Créer un robot Conversation Designer à partir d’un exemple de robot

Conversation Designer est une infrastructure puissante qui fournit un générateur de robots visuel, un modèle déclaratif pour la définition de robots et un runtime qui simplifie la création de bots. Une façon de simplifier la création de robot consiste à commencer à créer le robot à partir d’un robot existant.

Vous pouvez créer des robots Conversation Designer à partir de l’un des nombreux exemples de robots ou d’un robot Conversation Designer précédemment exporté. Chaque exemple de robot est entièrement fonctionnel. Cela vous donne un point de départ solide à partir duquel vous pouvez commencer à créer votre propre robot ou personnaliser l’exemple de robot pour l’adapter à vos scénarios.

## <a name="the-sample-bots"></a>Exemples de robots

Conversation Designer est fourni avec un ensemble d’**exemples de robots**. Vous pouvez commencer à créer votre robot à partir de l’un de ces **exemples de robots**. La plupart de ces **exemples de robots** couvrent un petit scénario de robot entièrement fonctionnel. Si vous souhaitez voir tous ces exemples de robots fonctionner ensemble, créez le **robot de Contoso Cafe complet**. 

Le tableau ci-dessous répertorie les types de robots que vous pouvez créer, avec une brève description de chacun d’eux et des scénarios qu’il couvre.

## <a name="explore-sample-bots"></a>Explorer les exemples de robots

Si l’exemple de robot que vous utilisez ne correspond pas tout à fait à ce que vous souhaitiez, vous pouvez basculer vers un autre exemple de robot à tout moment. Vous trouverez une liste de tous les exemples de robots dans la page **Explore sample bots** (Explorer les exemples de robots).

> [!WARNING] 
> L’importation d’un exemple de robot a pour effet de remplacer le robot actif par l’exemple de robot. Si vous ne souhaitez pas perdre votre personnalisation du robot actif, [exportez-le](conversation-designer-export-import-bot.md#export-a-conversation-designer-bot) dans un fichier .zip.

Pour basculer vers un autre exemple de robot, procédez comme suit :
1. Dans le robot actif, cliquez sur **Explore sample bots** (Explorer les exemples de robots). Cette option figure en bas du panneau de navigation à gauche. 
2. Choisissez un robot dans une liste d’**exemples de robots**, puis cliquez sur **Importer**.
3. Si vous souhaitez enregistrer votre travail dans le robot actif, choisissez l’option **Backup and import** (Sauvegarder et importer). Cette option a pour effet d’enregistrer le robot actif sur votre ordinateur local, puis d’importer le nouvel **exemple de robot**. Autrement, choisissez **Import without backing up** (importer sans sauvegarder).

Votre robot est remplacé par le nouvel exemple de robot que vous venez de sélectionner.
